Pagina de inicio de la aplicacion

Incorporar Bootstrap al proyecto:

  1. Acceder a https://getbootstrap.com/ y copiar el comando para instalar el modulo en el proyecto.
  2. En el proyecto abrir una terminal, pegar el comando y ejecutarlo
  3. En la carpeta node_modules buscar bootstrap, desplegar la carpeta dist y luego css y js.
  4. Dentro de css buscar el archivo bootstrap.min.css, clickear con el boton derecho y copiar la ruta relaiva.
  5. Buscar clave styles dentro del build y agregar la ruta copiada (va entre comillas " ", y de ser necesario reemplazar las barras \ por estas /).
  6. Volver a la carpeta js dentro de dist en node_modules y copiar la ruta relativa del archivo bootstrap.bundle.min.js
  7. Buscar clave scripts dentro del build y agregar la ruta copiada (va entre comillas " ", y de ser necesario reemplazar las barras \ por estas /).
